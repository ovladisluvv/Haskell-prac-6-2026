# Решение задач из курса весеннего практикума для студентов 3-го курса кафедры Алгоритмических языков ВМК МГУ

*В данном репозитории представлены решения домашних заданий практикума. Если вас интересует проектное большое задание, перейдите в следующий репозиторий: [Haskell Chess](https://github.com/ovladisluvv/Haskell-Chess-2026)*

## Структура курса:
*Получение и сдача заданий проходит на платформе Stepik.*

N небольших домашних заданий, направленные на реализацию алгоритмов или отработку навыков, полученных на семинаре. Перед отправкой работу программ можно тестировать прямо на сайте

*Важно учитывать: варианты выдаются согласно позиции студента в общем списке, отсортированном в лексикографическом порядке, однако почти все различия в разных вариантах одного и того же задания сугубо косметические*

## Структура репозитория

Задания из курса разбиты по папкам согласно следующему шаблону:
```text
    ├── taskN
    │ ├── taskN_1.hs
    │ │   ...
    │ └── taskN_M.hs
```
где N - номер домашнего задания, M - номер задачи в домашнем задании N
